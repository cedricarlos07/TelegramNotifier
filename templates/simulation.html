À propos du mode simulation

Le mode simulation vous permet de tester l'application sans envoyer de messages aux vrais groupes Telegram. En mode simulation, tous les messages seront envoyés uniquement au groupe de test spécifié ci-dessous. Les messages seront préfixés par [SIMULATION] pour les distinguer des messages réels.

Tous les messages en mode simulation seront envoyés à ce groupe Telegram. Assurez-vous que le bot a été ajouté à ce groupe avec les permissions nécessaires.
Comment utiliser le mode simulation
  1. Créez un groupe de test dans Telegram et notez son ID.
  2. Ajoutez votre bot Telegram à ce groupe avec les permissions d'administrateur.
  3. Activez le mode simulation et entrez l'ID du groupe de test ci-dessus.
  4. Exécutez vos scénarios normalement - tous les messages seront redirigés vers le groupe de test.
  5. Lorsque vous êtes prêt à utiliser le système en production, désactivez le mode simulation.
